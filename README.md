## 🚀 Deploy lên GitHub Pages
1. Tạo repo mới (Public) trên github.com
2. Upload `index.html` + `README.md` vào repo
3. Settings → Pages → Branch: **main** → Save
4. Truy cập: `https://USERNAME.github.io/REPO_NAME`
